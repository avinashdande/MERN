Selectors
---------
There are mainly 2 types of selectors, 
1 - Class selectors
2 - Id selectors

1 - Class selectors
-------------------
Used to maintain individuality in the HTML Elements 
Duplicates allowed (Same class can be used for several elements)
Class Selector starts with .(dot)

2 - Id Selectors
----------------
These serve same functionality as the class 
The main differenece is each Id should be Unique so, Every Element's Id is 100% different 
Id selector starts with #(Hashtag)


background property --> linear-gradient(90deg, red, green) 
	Angle and 2 colors are given for their combination !!

Styling
-------
Styling meant to applying styles to HTML tags so that the web page looks clean and great
We can apply styles in 3 methods they are,
	1 - Inline (Using style attribute)
	2 - Internal (Using style tag in the Head section
	3 - External (Using .css file)
<style></style> --> For Internal thing
<link rel="stylesheet" href=".css file"> --> For External usage

Images
------

To create in HTML we have to use <img> tag
<img> tag is a non-paired tag so, with the attributes we will display one
	Attributes
	----------
	1) width --> Used to define image width
	2) height --> Used to define image height 
	3) src --> Used to define image path (Where it's located)
	4) alt --> Used to display alternative text if img fails to load

TABLES
------

Tables are created using table tag and in that 2 sections will be present
1) thead --> For declaring the heading part (Optional)
2) tbody --> For declaring the body part (Optional)

Attributes
----------
	border --> Used to define border for a table
	cellpadding --> Used to add padding to each cell in the table
	cellspacing --> Used to add spacing between 2 cells that are adjancent (Simply, cell 2 cell gap)
	align --> center (All the elements will be aligned center 
Row Creation --> <tr> tag is used to create rows in the table
		 <th> tag is used to create heading of a column
		 <td> tag is used to create a column(data)
*** cellspacing = "0" will make the border linear
CSS styling for table
---------------------
	margin: auto --> Will make the table in the center 
	border-collapse: collapse --> Will make the borders of the table to linear(Common way)
	tr:nth-child(even) --> will select the rows that are even and we camn apply styles to them ******
Ex. tr:nth-child(even){
	background-color: orange;
	}
	Will make the even rows orange **

FORMS *** IMP ***
-----
Collection of related data is called a FORM 
Ex, Login, Enrollment, Registration etc...
	Tags on Form
	------------
	<form action="" method="" name="" onsubmit="" onreset="">
	There are mainly 5 most commonly used attributes
	1) action -> It is used to connect to backend
	2) method -> Represents the method that is used (GET/POST/PUT/DELETE)
		GET -- To read data from backend		
		POST -- Frontend wanted to send data backend
		PUT -- Frontend want to change edit/update backend data
		DELETE --  Frontend wanted to delete backend data
	3) name --> Used to assign logical name to the form
	4) onsubmit --> Used to handle submit operation/Event
	5) onreset --> Used to handle reset operation/Event 
<input> tag
----------
It is mainly used to create Textbox related stuff in HTML
Ex, <input type="textbox"/"number"/"tel"/"email"/"password"/"file"/"submit">
	Textbox -- For plain text
	Number -- For digits
	Tel -- For mobile number
	Password -- For password
	File -- For inputing a file 
	submit -- For submit button 
	Some other values are radio for Radio button, checkbox for checkbox square box 
Placeholder attrib is used for providing helper text to the user
